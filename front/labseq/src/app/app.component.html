Understanding the Labseq Sequence

The Labseq sequence is determined as follows:

  • l(0) = 0
  • l(1) = 1
  • l(2) = 0
  • l(3) = 1
  • For n > 3, l(n) = l(n-4) + l(n-3)

Example values: 0, 1, 0, 1, 1, 1, 1, 2, 2, 2, 3, ...
